# L3-Web-App-Project-AustinZhao

This is a NCEA level 3 application project built with Flask and SQLite.
The project is a video game rating and review database for user who want to compare games by title, price, and rating.

## Curent Sprint 1 Feature

- Flask application sturture
- SQlite database with sample game data
- Homepage that displays all games
- Search bar for finding games by title
- Top rated games pages
- Basic about page
- Responsive HTML and CSS layout
- 404 page
